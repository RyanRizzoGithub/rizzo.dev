Orbit: Rotate, Dodge, Survive

Introduction

Orbit is a fast-paced reflex game built with the HTML5 Canvas API. Players control a small orb that circles around a center point. With a tap of the spacebar, the direction of rotation flips — a simple mechanic that becomes increasingly difficult as enemies close in.

 

How the Game Works

Your player orb continuously moves in a circle around the center of the board. Each press of spacebar instantly reverses the direction. Pawns, robbers, and knights spawn from the edges and rush toward the center. Colliding with any of them ends the run.

How to Play

  • Spacebar → Reverse rotation direction
  • Click menu buttons to choose difficulty
  • Enter → Restart after game over

The longer you survive, the higher your score. Each difficulty increases enemy types and spawn rates. Try to beat your high score!

Obstacles & Enemies

  • Pawns → Slow, steady threats closing from all sides.
  • Robbers → Faster projectiles requiring quick reaction.
  • Knights → The deadliest enemies with blazing speed.

Canvas Rendering

Orbits’s smooth animations are powered by requestAnimationFrame. Enemies shrink the distance between themselves and the center while the player orb updates its circular position. 


      function update() {
          if (gameOver) return;
          frame++;
          angle += direction * 0.04;
          pawns.forEach(pwn => pwn.distance -= 2);
          robbers.forEach(rob => rob.distance -= 4);
          knights.forEach(knt => knt.distance -= 6);
      }

Game Flow

Players start at the menu screen by choosing a difficulty. Each mode alters the spawn rate and speed of enemies. The game ends on collision, displaying the current score and offering a chance to restart. 


      function restart() {
          cancelAnimationFrame(animationId);
          obstacles = [];
          robbers = [];
          knights = [];
          frame = 0;
          gameOver = false;
          score = 0;
          angle = 0;
          radius = 100;
          direction = 1;
          gameLoop();
      }

Coding Challenges

  • Managing movement & collision detection.
  • Balancing spawn rates across three enemy types.
  • Responsive difficulty scaling while keeping gameplay fun.
  • Ensuring smooth animations with requestAnimationFrame.

Final Thoughts

Orbit demonstrates how a single input mechanic can create deep, challenging gameplay. With simple controls and escalating difficulty, it captures the spirit of arcade reflex games while remaining easy to pick up and play.

Hexabine: A hexagonal twist on 2048

Introduction

Hexabine is a modern reinterpretation of the classic puzzle game 2048. Instead of a square grid, tiles live on a hexagonal board, which creates new movement mechanics and challenges. Players merge matching numbers by sliding tiles in six possible directions instead of four.

 

How the Game Works

Each turn, the player presses an arrow key(or diagonal combo of keys) to slide all tiles in that direction. If two tiles of the same value collide, they merge into a single tile with double the value. The objective is to reach ever higher numbers before the board fills up.

How to Play

Use the arrow keys to slide tiles across the hexagonal grid. You can also combine directions:

  • Arrow Left → Move Left
  • Arrow Right → Move Right
  • Arrow Left + Arrow Up → Move Up-Left
  • Arrow Right + Arrow Up → Move Up-Right
  • Arrow Left + Arrow Down → Move Down-Left
  • Arrow Right + Arrow Down → Move Down-Right

Tiles with the same number merge when they collide, forming a new tile with double the value. Plan your moves carefully — every turn spawns a new tile.

Objective

The goal of the game is to keep merging tiles until you create the 2048 tile (or higher!). Try to achieve the highest score you can before the grid fills up and no more moves are possible.

Board & Tiles

The game board is a 2D array of hexagonal positions. Each tile knows its value, grid coordinates, and pixel position on the canvas. 


      function spawnPiece() {
          const empties = [];
          for (let r = 0; r < tileHeight; r++) {
              for (let c = 0; c < tileWidth; c++) {
                  if (!grid[r][c]) empties.push([r, c]);
              }
          }
          if (empties.length === 0) return false;
          const [r, c] = empties[Math.floor(Math.random() * empties.length)];
          const value = Math.random() < 0.65 ? 2 : 4;
          makeTile(value, r, c);
          return true;
      }

Canvas Rendering

Rendering uses the HTML5 Canvas API. Tiles animate smoothly between positions using requestAnimationFrame. 


      function animate() {
          let anyMoving = false;
          for (let i = tiles.length - 1; i >= 0; i--) {
              const t = tiles[i];
              const targetP = gridToPixel(t.targetRow, t.targetCol);
              const dx = targetP.x - t.x;
              const dy = targetP.y - t.y;

              if (Math.abs(dx) > 0.5 || Math.abs(dy) > 0.5) {
                  t.x += dx * ANIM_SPEED;
                  t.y += dy * ANIM_SPEED;
                  t.moving = true;
                  anyMoving = true;
              } else {
                  t.x = targetP.x;
                  t.y = targetP.y;
                  t.moving = false;
                  if (t.willRemove) tiles.splice(i, 1);
                  else {
                      grid[t.targetRow][t.targetCol] = t;
                      t.row = t.targetRow; 
                      t.col = t.targetCol;
                  }
              }
          }
          draw();
          if (anyMoving) requestAnimationFrame(animate);
          else {
              animating = false;
              inputConsumed = false;
              spawnPiece();
              if (noMovesAvailable()) gameOver();
              draw();
          }
      }

Input & Movement

Unlike the original 2048, Hexabine supports six movement directions. Input is handled through arrow keys, with diagonal combinations triggering diagonal shifts. 


      const L = !!keysPressed['ArrowLeft'];
      const R = !!keysPressed['ArrowRight'];
      const U = !!keysPressed['ArrowUp'];
      const D = !!keysPressed['ArrowDown'];

      if (L && U) return 'UL';
      if (R && U) return 'UR';
      if (L && D) return 'DL';
      if (R && D) return 'DR';
      if (L) return 'L';
      if (R) return 'R';
      return '';

Game Flow

At the start of the game, two tiles spawn on the grid. After each move, a new tile appears. The game ends when no moves are available.

Coding Challenges

  • Managing six movement directions in a hexagonal grid.
  • Ensuring merges occur correctly and only once per move.
  • Smooth animations with requestAnimationFrame.
  • Handling multi-key input combos for diagonal movement.

Final Thoughts

Hexabine offers a refreshing take on a well-loved classic. Its hexagonal movement system encourages new strategies, while its clean canvas animations keep gameplay satisfying and intuitive.

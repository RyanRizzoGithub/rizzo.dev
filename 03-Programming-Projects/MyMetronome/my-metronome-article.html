MyMetronome: An interactive tempo and rhythm trainer with visual animations

Introduction

MyMetronome is an interactive rhythm tool I developed in JavaScript using the Web Audio API. It allows users to set beats per minute (BPM), time signatures, and toggle between multiple visual animations — including a pendulum and a horizontal oscillating ball — while maintaining perfectly synced audio cues.

This project is designed as both a practice aid for musicians and a demonstration of precise audio scheduling combined with dynamic DOM animation.

How It Works

The metronome relies on a scheduler function that uses the Web Audio API’s AudioContext to time clicks with high accuracy. Click tones are generated dynamically, allowing pitch changes for accented beats without preloading samples.

The program supports three animation modes:

  • Pendulum → A swinging pendulum synced to the beat.
  • Ball Oscillator → A ball moving horizontally in sync with the beats.
  • None → No animation for minimal distraction.

Key Challenges

Challenge 1: Precise Timing

Maintaining consistent rhythm is essential for a metronome. To solve timing inconsistencies caused by setInterval, I implemented a scheduling system using requestAnimationFrame and look-ahead calculations based on AudioContext.currentTime.

Challenge 2: Smooth Animation

Animating a pendulum or ball in perfect sync with the click requires dynamic transition calculations. I tied CSS transition durations directly to the BPM, ensuring visuals always match the audio rhythm.

Challenge 3: Dynamic Animation Modes

The toggle between pendulum, ball, and no animation required careful DOM structure design. This mode switching had to be smooth without interrupting the beat schedule.

Interactivity

Users can interact with the metronome in real-time, adjusting BPM and beats per measure, and switching animation modes without restarting. This makes it a powerful tool for musicians, music educators, and developers experimenting with audio-visual synchronization.

What Makes It Unique

Unlike static metronomes, this tool combines high-precision audio scheduling with visually engaging animations. The ability to toggle animation modes and control rhythm dynamically offers a tailored experience for practicing musicians.

Conclusion

The Dynamic Metronome is both a functional practice tool and an exploration of combining precise audio timing with modern visual animation techniques. It’s a project that merges music theory, web audio, and UI/UX design into a single interactive experience.
